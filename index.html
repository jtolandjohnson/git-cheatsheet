Your Name's Git Cheatsheet

Common commands

Initialize a folder into a new repo

git init

Add all new local files to be tracked in repo

git add .

Commit all your local changes to repo

git commit -a -m "commit message here"

Push your local changes to remote repo

git push

Pull remote changes to your local repo

Fetches remote changes

git fetch

Pull remote changes to your local repo

Fetches remote changes, then merges with your local repo

git pull

Get the status of your local repo compared to remote repo

Will show all changes not yet committed or pushed to remote repo

git status

Get the status of your local repo compared to last time you pulled

Will show all changes not yet committed or staged

git diff

Clone the code from a remote repo

git clone repourl

Create a branch in a repo

git branch branchname

Change to a branch in a repo

git checkout branchname

Stash/disregard your local changes

git stash
